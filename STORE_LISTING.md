# Chrome Web Store listing

## Name

Explain This – Instant AI Explanations

## Short description

Understand highlighted text instantly without leaving the page.

## Headline

50 free AI explanations every day. No API key required.

## Full description

Read without losing your flow.

Explain This turns highlighted words, phrases, and passages into clear AI
explanations while you stay on the page. Choose a concise answer when you only
need the core meaning, or a detailed explanation with key points and an example.

What you get:

- 50 free AI explanations every day
- No account or API key required
- English and Chinese explanations
- Concise and detailed answer styles
- Optional recent history and local cache
- Keyboard shortcut: Ctrl+E or Cmd+E
- Bring Your Own Key in Advanced settings for higher limits

Privacy-conscious by design:

- Only text you actively submit is processed
- No browsing-history tracking
- No free-mode model key inside the extension
- Selection tools run only when you invoke the extension or allow a site
- Enabled-site permissions can be reviewed and removed at any time
- Recent explanations can be disabled and cleared

Explain This is an explanation layer for the web—not another search tab.
